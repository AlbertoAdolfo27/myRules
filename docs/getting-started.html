Quick start

Copy-paste the following script into your web page. If you need, you can download the source file here.

<script src="http://exemplo.cdn/myrules.min.js"><script/>

Basic example

To validate a form element with My Rules, add a class .mr into the element's class and add the class or attribute of validation (eg. class .mr-required or attribute required). You will see more classes and attributes used by My Rules to validate an element in this documentation.

Is needed an an .mr-feedback for each .mr element to display the validation feedback. For each .mr element has the correspondent feedback's class.

Eg. the feedback class of .mr-required is .mr-required-fb. To display the feedback of a required element you add the class .mr-feedback and mr-required-fb into the feedback element's class.

Try to submit the form without fill out the text fields.

<!doctype html>
<html lang="en">
    <head>
        <title>Hello, world!</title>
        <meta charset="utf-8">
    </head>
    <body>
        <form>
            <!--Required text field-->
            <input type="text" placeholder="First name" class="mr" required> <br>
            <!--Validation feedback-->
            <p class= "mr-feedback mr-required-fb">Please fill out this field</p>

            <!--Required text field-->
            <input type="text" placeholder="Last name" class="mr mr-required"> <br>
            <!--Validation feedback-->
            <p class= "mr-feedback mr-required-fb">Please fill out this field</p>

            <input type="submit" value="submit">
        </form>

        <!--MY RULES-->
        <script src="http://exemplo.cdn/myrules.min.js"></script>
    </body>
</html>

Validate element on data input

By default the form element will be validated only on element value change and on form submit, also to validate the form element when user input data, add a class .mr-oninput to your <form>.

Try to input an invalid integer number.

Note: some browsers can ignore illegal characters from the input type number.


Please input a valid integer number

<form class="mr-oninput">
    <input type="number" class="mr mr-number-int"> <br>
    <p class="mr-feedback mr-number-int-fb">Please input a valid integer number</p>

    <input type="submit" value="submit">
</form>

My Rules customized colors

To use My Rules customized colors on valid and invalid form elements, you must add a class .mr-colors on your <form>.

Try to input invalid alpha and space (not between [A-Za-z ]) in the first text field and invalid alpha (not between [A-Za-z]) in the second text field.


Please input a valid first name


Please input a valid Last name

<form class="mr-colors">
    <input type="text" placeholder="First name"  class="mr mr-alpha-space"> <br>
    <p class="mr-feedback mr-alpha-space-fb">Please input a valid first name</p>

    <input type="text" placeholder="Last name" class="mr mr-alpha"> <br>
    <p class="mr-feedback mr-alpha-fb">Please input a valid Last name</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
If you want, you can customize the form element when it's valid or invalid yourself by using the CSS class selector of each validation status, .mr-valid when the element is valid and .mr-invalid when it's invalid. Also you can custom the feedback elements by using its classes.

Multiple validation

You can have more than one validation in the some form element. To display validation feedback of an element with multiple type of validation you must create one <div> with .mr-feedback and add elements with feedback's class inside them.

  • The valid username characters are uppercase, lowercase, number, underscore, dot and at least two characters.
  • The strong password must have an uppercase, a lowercase, a number and/or a special char and at least 6 characters.
  • The Both text fields are required.

<form class="mr-oninput mr-colors">
    <input type="text" placeholder="Username" class="mr mr-username" required> <br>
    <div class="mr-feedback">
        <p class="mr-required-fb">Please fill out this field</p>
        <p class="mr-username-fb">Please input a valid username</p>
    </div>

    <input type="password" placeholder="Password" class="mr mr-password-strong" password-minlength="6" required> <br>
    <div class="mr-feedback">
        <p class="mr-required-fb">Please fill out this field</p>
        <p class="mr-password-strong-fb">The password must have an uppercase, a lowercase, a number and/or a special char and at least 6 characters</p>
        <p class="mr-valid-fb">Strong password</p>
    </div>

    <input type="submit" value="submit">
</form>
